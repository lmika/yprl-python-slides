Step 1: Booleans

Dealing With Truth

Types So Far

  • Strings: Sequence of characters
  • Integers: Numbers without fractional parts
  • Floats: Numbers with fractional parts

Today, we'll introduce a new one

Booleans

  • Based on boolean logic
  • Only two values:
    • True: something is the case
    • False: something is not the case
  • If something is False, it is not True (and visa versa).

Used for comparison and branching

==: Equals Operator

Checks to see if two values are equal to each other.

3 == 3               # True
5.3 == 5.3           # True
"Hello" == "Hello"   # True
False == False       # True

Values must be of similar types.

3 == "3"             # False
"False" == False     # False

!=: Not Equals Operator

Checks to see if two values differ.

3 != 4               # True
5.3 != 8.4           # True
"Hello" != "World"   # True
False != True        # True

If types are not similar, returns true:

3 != "3"             # True
"0" != 0             # True

> and >=: Greater Than

> checks to see if the first value is greater than the second.

4 > 3                # True
2.1 > 7              # False
6 > 6                # False

>= is like > and ==:

4 >= 3               # True
2.1 >= 7             # False
6 >= 6               # True

< and <=: Less Than

< checks to see if the first value is less than the second.

4 < 3                # False
2.1 < 7              # True
6 < 6                # False

<= is like < and ==:

4 <= 3               # True
2.1 <= 7             # False
6 <= 6               # True

Comparison Operators

Operators which compare two values and returns a boolean:

  • ==: True if two values are equal
  • !=: True if two values are not equal
  • >: True if the first value is greater than the second
  • >=: True if the first value is greater than or equal to the second
  • <: True if the first value is less than the second
  • <=: True if the first value is less than or equal to the second

Step 2: The If Statement

Running Code Some of the Time

Branching

Every statement in the program is executed.

Sometimes we want to execute statements when a condition is true.

The if Statement

if (guess == 5):
    print("Displayed only if the value of the")
    print("guess variable equals 5")
print("Always displayed")
  • The condition within the brackets is a boolean expression
  • If it is true, the statements within the block are executed.
  • If it is not true, the statements within the block are skipped.

The block is any code to the right of the if statement.

The if Statement

The statements to the right of if only runs if the condition is true.

The else Statement

if (guess == 5):
    print("Displayed only if the value of the")
    print("guess variable equals 5")
else:
    print("Displayed only if the value of the")
    print("guess variable does NOT equal 5")
  • Comes after the block of an if statement.
  • Only executed only if the condition is not True (i.e. False).

The else Statement

The statements to the right of else only runs if the condition is false.

Step 3: The While Statement

Running Code More Than Once

Looping

  • The if statement allows us to skip some code
  • Sometimes it is necessary to go back and rerun some code

A loop is a way to run some code more than one time

The while Loop

while (guess < 10):
    print("Will keep displaying this")
    print("while guess is less than 10")
    guess = guess + 1
print("Displayed once guess >= 10")
  • The condition within the brackets is a boolean expression
  • While it is true, the statements within the block are executed.
  • Until it is not true, the statements within the block are skipped.

The while Loop

The statements to the right of while will keep running while the condition is true.

Infinite Loops

  • The while loop condition is checked every time

  • If the condition can never be False, the loop will never end:

    while (guess < 10):
    print("Guess is < 10")
    # The "guess" variable is never changed so
    # the loop will never end
print("This is never displayed")

  • To avoid, make sure that the variables within the condition change

Summary

  • A boolean can either be True or False
  • An if statement executes a block if the condition is True
  • An else statement can be added to an if. It will run if the condition is False.
  • A while loop will keep running a block while the condition is True.
  • A block is any code to the right of an if/else/while statement.
