Numerical Operators

Takes two numbers and produces another number

  • +: Add two numbers
  • -: Subtract two numbers
  • *: Multiply two numbers
  • /: Divide two numbers

These operators can be used to formulate mathematical expressions:

print(2 * (5 - 7) + 12)
8

Variables

  • Reference to memory to store a value

  • Identified by a name: birthYear

  • XXX Picture

Variables

  • Values are stored in (or assigned) to a variable using the = operator:

    birthYear = 1985        # Variable on left, value on right

  • When using a variable name in place of a value, the variable's value will be used instead:

    2025 - birthYear

  • Variables can also be set from the value of other variables (or even itself)

    age = 2025 - birthYear
count = count + 1

Step 3: Input

Accepting Input From The User

The Importance Of Input

"Variables are great, but we're still changing the program if we want to test for a particular year"

  • We've used variables to store constant values within our program
  • Now, we'll use variables to store values coming outside our program

We'll learn how to ask the user for values

The input() Function

input(prompt) - displays prompt, reads a line of input, converts it to a string, and returns that

    :::python
    year = input("What year were you born? ")
    age = 2025 - year

What happens when we try that?

TypeError: unsupported operand type(s) for Sub: 'int' and 'str'

What happened? We're mixing a number with a string

Numerical operators take two numbers and produce another number

Need to find a way to convert a string into a number...

The int() Function

int(x) - returns an integer constructed from a number or a string

    :::python
    year = input("What year were you born? ")
    age = 2025 - int(year)
