Arithmetic Operators

Operators Revisited

  • We've seen operators that work on strings:

    "Hello " + "world" = "Hello world"
"Three " * 3 = "Three Three Three"

  • Python has operators that work on other types

But what is a type?

Types

  • Classify what some data represents
  • The language defines certain properties and rules for each type
    • How they look in the program
    • How they are stored in memory
    • What operations can be performed on them
  • Every value in Python has a type:
    • Integers: 123, -12
    • Floating points: 15.0, 3.14
    • Strings: "Hello world", 'Another string'
  • Type defines

Numerical Operators

Takes two numbers and produces another number

  • +: Addition
  • -: Subtraction
  • *: Multiplication
  • /: Division

These operators can be used to formulate mathematical expressions:

print(2 * (5 - 7) + 12)  # => 8

Step 2: Variables

Storing Values In Memory

The Importance Of Memory

print("Since you were born in", 1985)
print("you will be", 2025 - 1985, "in 2025.")
  • What needs to be done if we want to change any of the years used?
  • What if we need to run this for 30 different people, each born in a different year?
  • What if we want to ask the user first?

Need to allow for values to change without changing the code

Variables

  • Reference to memory to store a value
  • Identified by a name: birthYear

Naming Variables

  • Variable names can have any combination of:
    • Letters: age, DATE, birthDay
    • The underscore: a_long_variable_name
  • Variable names can have numbers, but not at the front: abc123
  • Variable names are case sensitive:
    • birthYear is different to Birthyear

Variables

To use a variable, set it to a value by using the assignment operator:

birthYear = 1985        # Variable on left, value on right

Variables

When using a variable name in place of a value, the variable's value will be used:

2025 - birthYear

Variables can also be set from the value of other variables (or even itself)

age = 2025 - birthYear
count = count + 1

Step 3: Input

Accepting Input From The User

The Importance Of Input

"Variables are great, but we're still changing the program if we want to test for a particular year"

  • We've used variables to store constant values within our program
  • Now, we'll use variables to store values coming outside our program

We'll learn how to ask the user for values

The input() Function

input(prompt) - displays prompt, reads a line of input, converts it to a string, and returns that

year = input("What year were you born? ")
age = 2025 - year

What happens when we try that?

TypeError: unsupported operand type(s) for Sub: 'int' and 'str'

What happened? We're mixing a number with a string

Numerical operators take two numbers and produce another number

Need to find a way to convert a string into a number...

The int() Function

int(x) - returns an integer constructed from a number or a string

year = input("What year were you born? ")
age = 2025 - int(year)
