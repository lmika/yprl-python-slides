Step 2: ASCII Art

Anatomy of a Program

Programs Are a Series Of Steps

  • Programs are made up of instructions (e.g. print).
  • These instructions are called statements.
  • Each statement tells the computer to do something.
  • When we run the program, the statements will be executed one after the other.

But, how do we write these instructions?

  • 2: write something to the screen, read input, perform calculation, run a block

Programming Language

  • Programming Languages are a set of rules on how to write instructions that can be understood by the computer.
  • Many, many different types of programming languages out there.
  • The difference between them is how they let you express what you want the computer to do.
  • 1: how to name things, represent values, blocks and modules
  • 2: Python, C, Java, Ruby, Swift, Go, C++, JavaScript, Perl, Scala, Lisp, Prolog, Haskell, Lua

Step 3: Calculating Text

Operators

Operators

  • Fundamental way of manipulating data
  • Operators work like mathematical expression:
    • Take two (or one) pieces of data: the operands
    • Processes them to produce another piece of data: the result
  • Today: look at operands that will produce text (or strings) as the result

Operating On Strings

What do we get if we:

  1. Multiply "hello" by 5?

    print("hello" * 5)

  2. Multiply "hello " by 5?

    print("hello " * 5)

  3. Add "hello " to "world"?

    print("hello " + "world")

  4. Subtract "world" from "hello world"?

    print("hello world" - "world")
