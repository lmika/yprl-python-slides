Step 2: ASCII Art

Anatomy of a Program

Programs Are a Series Of Steps

  • Programs are made up of instructions, called statements
  • Each statement tells the computer to do something:
    • Write something to the screen (e.g. print())
    • Read input from the keyboard
    • Perform some calculation
    • Store something in memory
    • Do a set, or block, of statements if some other conditions are met
  • When we run the program, the statements will be executed one after the other.

But, how do we write these instructions?

Programming Language

  • Programming Languages are a set of rules on how to write instructions that can be understood by the computer.
    • How to name things
    • How to represent values like text and numbers
    • How to write programs with more than one instruction
    • How to structure large, multi-file programs
  • Many, many different types of programming languages out there:
    • Python, C, Java, Ruby, Swift, Go, C++, JavaScript, Perl, Scala, Lisp, Prolog, Haskell, Lua...
  • The difference between them is how they are converted into instructions the computer understands

The Secret Of Computers

  • Computers are dumb
    • They need to be explained everything (print?! What's that?)
  • Computers are fast
    • 2.0 GHz = 2,000,000,000 ops per second
  • We use the speed of computers to make it easier for us to tell them what to do.
  • To do that, we use programs designed to understand and execute other programs.

Step 3: Calculating Text

Operations

Operations

  • Fundamental way of manipulating data
  • Operations work like mathematical expression:
    • Take two (or one) pieces of data: the operands
    • Processes them to produce another piece of data: the result
  • Today: look at operands that will produce text (or strings) as the result

Operating On Strings

What do we get if we:

  1. Multiply "hello" by 5?

    print("hello" * 5)

  2. Multiply "hello " by 3?

    print("hello " * 3)

  3. Add "hello" to " world"?

    print("hello " + "world")

  4. Subtract "world" from "hello world"?

    print("hello world" - "world")

Switch to the console to try this out.
